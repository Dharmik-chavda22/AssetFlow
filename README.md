# Overview

AssetFlow is a modern Enterprise Asset & Resource Management System designed to help organizations efficiently manage their physical assets, shared resources, maintenance workflows, and audit cycles from a centralized platform.

The system eliminates manual asset tracking through spreadsheets and paper logs by providing a secure, role-based, scalable ERP solution.

This project is built for organizations such as:

- Corporate Offices
- Educational Institutions
- Hospitals
- Manufacturing Industries
- Government Organizations
- Startups
- Warehouses

---

# Problem Statement

Organizations often struggle with:

- Manual asset tracking
- Duplicate asset allocation
- Missing assets
- Maintenance delays
- Resource booking conflicts
- No centralized reporting
- Poor audit management

AssetFlow solves these challenges by providing a centralized ERP platform with complete asset lifecycle management.

# Installation

Clone Repository

```bash
git clone https://github.com/yourusername/assetflow.git
```

Go to Project

```bash
cd assetflow
```

Install Dependencies

```bash
npm install
```

Run Project

```bash
npm run dev
```

Build Project

```bash
npm run build
```
